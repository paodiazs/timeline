Línea del tiempo

Paola Díaz Serrano

Programación de dispositivos móviles

2008

Lanzamiento de iOS y la App Store

La introducción del iPhone y la App Store de Apple marcó un punto de inflexión en la programación móvil,
creando un ecosistema próspero para desarrolladores y usuarios por igual.

2009

Introducción de las aplicaciones híbridas y PhoneGap

PhoneGap (ahora conocido como Apache Cordova) permitió a los desarrolladores crear aplicaciones móviles utilizando tecnologías web estándar como HTML, CSS y JavaScript, lo que facilitó la creación de aplicaciones para múltiples plataformas.

2010

Adopción de Dark Mode

Muchas plataformas móviles han implementado el modo oscuro (dark mode) en sus sistemas operativos y aplicaciones, brindando a los usuarios una opción visual alternativa y reduciendo el consumo de energía en pantallas OLED.

2014

Backend de Servicios Móviles

uchas aplicaciones móviles dependen de servicios de backend para funciones como autenticación, almacenamiento de datos, notificaciones push, análisis, etc. Estos servicios backend a menudo se pueden implementar en contenedores dentro de un clúster de Kubernetes, lo que permite una administración más eficiente y escalabilidad según la demanda.

2014

Lanzamiento de Swift

Apple introdujo Swift, un nuevo lenguaje de programación, como alternativa a Objective-C para el desarrollo de aplicaciones iOS y macOS. Swift se diseñó para ser más moderno, seguro y eficiente.

2015

Lanzamiento de React Native

React Native, desarrollado por Facebook, revolucionó el desarrollo de aplicaciones móviles al permitir a los desarrolladores construir aplicaciones nativas utilizando JavaScript y React. Esto permitió compartir código entre plataformas iOS y Android.

2017

Adopción de Kotlin como lenguaje oficial para Android

Google anunció que Kotlin se convirtiera en el segundo lenguaje oficial para el desarrollo de aplicaciones Android, ofreciendo una alternativa moderna y más segura a Java.
